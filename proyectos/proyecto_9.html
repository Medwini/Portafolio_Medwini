Descripción del proyecto

El proyecto consiste en un evaluador de texto con ayuda de una IA llamada Hugging Face, dónde puede determinar las emociones o sentimientos del texto. El FrontEnd se encuentra desarrollado con ReactJS, el cuál lee los datos de una API Rest mediante el uso de Fetch. El BackEnd se encuentra desarrollado con Django Rest Framework y consiste en una API capaz de registrar, consultar, editar y/o eliminar datos. El usuario podrá crear un usuario con su contraseña, iniciar sesión, cargar archivos csv en el cuál debe estar el texto, una vez que cargue el archivo, estos datos se guardan en la base de datos y se muestra la pantalla de Mostrar Datos, dónde se podrá observar los datos guardados y el resultado obtenido con la IA sobre la emoción y sentimiento del mensaje.

El proyecto fue desarrollado en Visual Studio Code, con ReactJS y CSS.

Link de la página: https://proyecto-evaluador-de-texto.netlify.app/
